# new-mall
这是一个移动端的商城项目，使用vue构建项目，vant ui框架，项目api未删除，可正常使用
# 使用方式
下载压缩包到电脑，在cmd中npm i 下载包， 然后使用npm run serve命令即可
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
